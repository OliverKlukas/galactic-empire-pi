# galactic-empire-pi
Reverse engineered implementation of Galactic Empire for C64 that natively runs on a raspberry pi.

# instructions
```bash
cl65 -O galacticEmpire.prg galacticEmpire.c -t c64
x64sc galacticEmpire.prg
```
